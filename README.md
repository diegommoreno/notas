# notas
practica notas
Notas es una aplicación simple de escritorio desarrollada en **C# (Windows Forms)**.  
Su propósito es permitir al usuario crear, editar y guardar notas de manera rápida y sencilla.

## Características
- Crear nuevas notas  
- Editar notas existentes  
- Eliminar notas  
- Interfaz minimalista y fácil de usar  

## Tecnologías utilizadas
- C#  
- Windows Forms  
- .NET Framework  

## Cómo ejecutar
1. Abrir el proyecto en **Visual Studio**.  
2. Compilar (`Ctrl + Shift + B`).  
3. Ejecutar (`F5`).
